# LoLLMs WebUI

**LoLLMs** (Lord of Large Language and Multimodal Systems) is an open-source web interface for interacting with large language models locally or via APIs.

## Installation

```bash
git clone https://github.com/ParisNeo/lollms-webui
cd lollms-webui
# Windows
install.bat
# Linux / macOS
bash install.sh
```

See [ManualInstall.md](ManualInstall.md) for detailed cross-platform instructions.

## Limitations & Risks

- Outputs from language models may be inaccurate, biased, or inappropriate. Always review generated content before use.
- LoLLMs WebUI is a tool that relies on third-party models; the quality and safety of responses depend on the underlying model used.
- Users are responsible for complying with the terms of service of any model or API they connect to.
